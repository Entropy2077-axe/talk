# GitHub Social Preview 设置

仓库已经生成 `docs/assets/talk-social-preview.png`（1280×640）。GitHub 暂无稳定的仓库 Social Preview 上传 API，因此需要仓库所有者完成一次手动设置：

1. 打开仓库 **Settings → General**。
2. 找到 **Social preview**。
3. 点击 **Edit → Upload an image**。
4. 上传 `docs/assets/talk-social-preview.png` 并保存。

之后在聊天软件、V2EX、即刻等平台分享仓库链接时会优先显示这张图。
