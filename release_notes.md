## 世界书集合与导入

- 世界书改为集合式管理，可独立启用、重命名、编辑和删除
- 支持导入 SillyTavern 世界书与角色卡、NovelAI、Risu、Agnai 及通用 JSON
- 新增底层世界观、常驻条目、关键词条目和 Token 占用估算
- 捏人时可从任意集合选择仅用于本次生成的额外世界观

## 捏人与好感度

- 普通模式支持自动计算或自定义初始好感度
- 女娲模式由 AI 生成初始好感度，并允许用户在初稿中再次修改
- 已保存的人设支持安全删除，不影响已创建联系人和永久创建历史

## 仓库与聊天体验

- 相同仓库物品自动合并数量，赠送至零后显示“已用完”并支持原价复购
- 修复手机端触摸抖动导致长按菜单和“重新生成这一轮”无法打开的问题
- 首次进入聊天时提示长按消息可用的功能
- 改进部分模型对 JSON 转换、格式审查请求的兼容性
